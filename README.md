# Safety-Count
SafetyCount คือเว็บแอปพลิเคชันจัดการความปลอดภัยและเช็คชื่อพนักงานแบบครบวงจร รองรับการเช็คชื่อเข้างานตอนเช้าโดยหัวหน้าแผนก และระบบรายงานตัวฉุกเฉินเมื่อเกิดเหตุภัยพิบัติ พร้อมฟังก์ชันลับในการตรวจสอบยอดข้ามกับข้อมูลการแตะบัตร (Badge Swipe) ช่วยให้องค์กรทราบจำนวนพนักงานที่แม่นยำแบบเรียลไทม์ ป้องกันคนสูญหาย เพื่อความปลอดภัยสูงสุดของทุกคน
